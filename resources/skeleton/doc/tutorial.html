JSesh Programming Tutorial

Welcome to JSesh programming tutorial. This documents explains how to use JSesh as a library to use in your own programs. If you are looking for information about how to use JSesh as a hieroglyphic editor, look here instead.

This document contains two parts. The first is a simple, do-by-example explanation of how to use jsesh, focusing on the most likely usages of the library. The second is an in-depth explanation of jsesh internals, for those who want to extend the library, or are simply interested in its architecture.
